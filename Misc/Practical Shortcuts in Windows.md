# Practical ShortCuts In Windows

## board terminal
- open/minimiz window `win` + `\``
- command: `ctrl` + `shift` + p

- `win` + s: search
- `win` + a: manage notify
- `win` + e: open file manager
- `win` + d: hide/display windows
- `win` + w: whiteboard and screenshot

- `win` + `shift` + s: screenshot
